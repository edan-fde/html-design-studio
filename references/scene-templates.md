## 1. WeChat Official Account Cover / Article Hero Image

**Specifications**:
- Cover image: 2.35:1 (900×383px or 1200×510px)
- Text illustration: 16:9 (1200×675px) or 4:3 (1200×900px)

**Key design elements**:
- Prioritize immediate visual impact; users scan quickly in the feed
- Little or no text (the article title will be overlaid separately)
- Moderate color saturation (WeChat reading environment is white)
- Avoid excessive details (thumbnails must also be legible)

**Recommended style**: 01 Pentagram / 11 Build / 12 Sagmeister / 18 Kenya Hara / 07 Field.io

## 2. Article Illustrations / Concept Visuals

**Specifications**:
- 16:9 (1200×675px) is the most versatile
- 1:1 (800×800px) is suitable for emphasis
- 4:3 (1200×900px) is suitable for dense information

**Key design elements**:
- Serves the argument of the article, not decoration
- Creates visual rhythm within the surrounding article
- Concisely expresses a core concept
- Prefer AI-generated visuals; use HTML screenshots only when tables or data must be exact

**Recommended style**: Match the tone of the article; common choices are 01/04/10/17/18

**Scene prompt template**:
```
[Insert style DNA here]
- Article illustration, concept visualization
- [16:9 / 1:1 / 4:3] aspect ratio
- Single clear concept: [describe the core concept]
- Serve the argument, not decoration
- [Light/Dark] background to match article tone
```

## 3. Infographics / Data Visualization

**Specifications**:
- Vertical: 1080×1920px for mobile reading
- Horizontal version: 1920×1080px (embedded in the article)
- Square: 1080×1080px (social media)

**Key design elements**:
- Clear information hierarchy (title → core data → details)
- Accurate data, no fabrication
- A clear visual path for the reader's eye
- Appropriate use of icons/charts to assist understanding

**Recommended style**: 04 Fathom / 10 Müller-Brockmann / 02 Stamen / 17 Takram

## 4. PowerPoint / Keynote presentation

**Specifications**:
- Standard: 16:9 (1920×1080px)
- Widescreen: 16:10 (1920×1200px)

**Key design elements**:
- One core message per slide (do not stack unrelated ideas)
- Clear font size hierarchy (title 40pt+ / text 24pt+ / annotation 16pt+)
- Generous whitespace for projection clarity
- Image-to-text ratio of at least 60:40
- Consistent visual system (color, font, spacing)

**Recommended style**: 01 Pentagram / 10 Müller-Brockmann / 11 Build / 18 Kenya Hara / 04 Fathom

**Scene prompt template**:
```
[Insert style DNA here]
- Presentation slide design, 16:9
- One core message per slide
- Clear type hierarchy (title 40pt+, body 24pt+)
- Generous whitespace for projection clarity
- Consistent visual system throughout
- [Light/Dark] theme
```
