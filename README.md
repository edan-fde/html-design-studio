# Dependencies

## Required

- Node.js 20+
- npm / npx
- Python 3.10+
- Playwright for Node.js
- Playwright for Python
- Chromium
- pdf-lib
- pptxgenjs
- sharp

```bash
npm ci
npx playwright install --with-deps chromium
python3 -m pip install playwright
python3 -m playwright install chromium
```

## Optional

- ffmpeg / ffprobe
- HyperFrames
- LibreOffice / PowerPoint
- requests / uv
- Doubao API
- ARK API
